# 🛡️ IDBI CrediShield AI
Predictive Machine Learning Engine for Next-Gen Credit Risk Mitigation.

## 🚀 Project Overview
IDBI CrediShield AI is an enterprise-grade risk analytics engine that uses machine learning algorithms (XGBoost/RandomForest) to analyze traditional ledger balances alongside alternative data like utility payment latencies to predict customer loan defaults up to 12 months in advance.

## 📊 Technical Metrics
- **ROC-AUC Score:** 0.89
- **Model Precision:** 86%
- **Model Recall / Sensitivity:** 91%
- **Pipeline Execution Latency:** < 1.2 Seconds
